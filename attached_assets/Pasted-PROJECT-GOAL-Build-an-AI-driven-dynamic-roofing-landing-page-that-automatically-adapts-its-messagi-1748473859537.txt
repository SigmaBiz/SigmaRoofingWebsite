PROJECT GOAL:
Build an AI-driven, dynamic roofing landing page that automatically adapts its messaging based on trending roofing-related Google searches in Oklahoma. The system should:
1. Fetch search phrases from Google Trends using pytrends
2. Parse phrases into structured variables (like city, storm type, hail size)
3. Save these entries in a JSON file or simple DB with unique phrase_id keys
4. Render a landing page dynamically based on ?phrase_id=X

TECH STACK:
- Python (backend script with pytrends)
- JSON file or Replit DB (to store phrase metadata)
- HTML/CSS/JS (frontend for landing page rendering)
- JavaScript (to extract phrase_id from URL and populate page)
- Optional: Replit cron job or manual trigger to update phrase data
